## Run the Frontend

Keep the backend terminal running.

Open a second terminal in the project root and run:
```
python -m http.server 5500 --directory frontend
```
Open:

```
http://127.0.0.1:5500
```

Then:

- select Data/sample_onchain_transactions.csv;
- click Analyze Data;
- wait for the backend analysis;
- view Data Quality;
- view Growth Metrics;
- view User Segmentation;
- view Wallet Preview.
